# PentAIGen - Your Financial AI Team

A comprehensive financial management platform powered by AI, providing real-time insights, fraud detection, investment tracking, and personalized financial advice.

## 🚀 Features

- **Dashboard**: Real-time financial overview with interactive charts and metrics
- **Investments**: Portfolio tracking, performance analysis, and asset allocation
- **Net Worth**: Comprehensive wealth tracking with historical trends
- **Fraud Alerts**: AI-powered transaction monitoring and security alerts
- **Tax Optimizer**: Smart tax planning and deduction tracking
- **Ask AI**: Personalized financial advice and insights
- **Mobile Responsive**: Fully responsive design for all devices

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Charts**: Recharts
- **State Management**: React Query (TanStack Query)
- **Routing**: React Router DOM
- **Icons**: Lucide React

## 📱 Mobile Optimized

The application is fully responsive with:
- Adaptive layouts for mobile, tablet, and desktop
- Touch-friendly navigation
- Optimized charts for smaller screens
- Mobile-first design approach

## 🚀 Deployment

This project is configured for deployment on Vercel:

### Prerequisites
- Node.js 18+
- npm or yarn

### Local Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Vercel Deployment
1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect the Vite framework
3. The build process will use `npm run build`
4. Your app will be deployed at your Vercel domain

## 🔧 Configuration

- **Build Output**: `dist/` directory
